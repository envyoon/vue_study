### Vue 설치 방법

1. npm install -g vue
2. npm install -g @vue/cli

### Vue 버전 확인

1. vue --version

만약 버전확인이 안될 경우 Windows PowerShell을 관리자 권한으로 실행하고

```
Set-ExecutionPolicy Unrestricted
A
```

를 입력한다.

### Vue 버전 업데이트

npm update -g @vue/cli
